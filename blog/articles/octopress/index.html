Octopress

dev Feb 19th, 2014

A blogging framework for hackers. を掲げるブログシステム。Jekyll のラッパー。So! アイツはマダコだ Yo!

http://octopress.org

Install

$ git clone git://github.com/imathis/octopress.git octopress
$ cd octopress
$ bundle install
$ rake install

Configuration

_config.yml を編集。
各項目は Configuring Octopress を参照。

Usage

記事作成

$ rake new_post['title']

source/_posts/ に YYYY-MM-DD-title.markdown が作成されるので、let’s まーくだうん。

ページ作成

$ rake new_page['title']

source/ に title/index.markdown が作成されるので、enjoy まーくだうん。

ウェブサイト生成

$ rake generate

わーい。

プレビュー

$ rake preview

で、WEBrick が起動するので http://localhost:4000 から確認。
Thin を使いたければ Gemfile へ追記を。

GitHub Pages

設定

$ rake setup_github_pages

で、リポジトリ URL を設定。git@github.com:username/username.github.io.git みたいなものを。

デプロイ

$ rake generate
$ rake deploy

or

$ rake gen_deploy

でおしまい。

カスタムドメイン

ドメインを記載した source/CNAME を作って、DNS の A レコードを編集。
A レコードの値は Setting up a custom domain with Pages を参照。
