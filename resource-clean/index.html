resource_clean

Oct 14, 2014

Delete resource file and xattr

http://rubygems.org/gems/resource_clean

  • .AppleDouble
  • .DS_Store
  • .fseventsd
  • .Spotlight-V100
  • .TemporaryItems
  • .Trashes
  • .VolumeIcon.icns
  • Desktop.ini
  • Thumbs.db

and, "._" prefix file

Installation

Add this line to your application's Gemfile:

gem 'resource_clean'

And then execute:

$ bundle

Or install it yourself as:

$ gem install resource_clean

Usage

$ resource_clean [directory]
